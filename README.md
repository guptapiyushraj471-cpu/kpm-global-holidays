### âš™ï¸ Installation

**Option 1: Static Website (HTML/CSS/JS only)**

```bash

# 2. Navigate to the project directory
cd kpm-global-holiday

# 3. Open index.html in your browser
# Double-click index.html or use:
# - Windows: start index.html
# - macOS: open index.html
# - Linux: xdg-open index.html
```

**Option 2: With Node.js/Express Backend**

```bash
# 1. Clone the repository
git clone https://github.com/yourusername/kpm-global-holiday.git

# 2. Navigate to the project directory
cd kpm-global-holiday

# 3. Install dependencies
npm install

# 4. Set up environment variables
cp .env.example .env
# Edit .env with your configuration

# 5. Start the server
npm start
```

### ğŸƒ Running the Project

**Development Mode:**

```bash
# With live reload (if configured)

```

**Production Mode:**

```bash
# Build and serve
npm run build
npm start
```

**Using a Simple HTTP Server:**

```bash

# Using Node.js http-server
npx http-server -p 8000

# Visit: http://localhost:8000
```
